#  Issue Tracker (Rails + React)

A simple full-stack Issue Tracker application built with **Ruby on Rails (backend)** and **React (frontend)**.

It allows users to create, view, and delete issues.

---

##  Tech Stack

- Backend: Ruby on Rails (API)
- Frontend: React (Vite)
- Database: SQLite (default Rails DB)
- API: REST APIs (JSON)

---

##  Features

- Create new issues
- View all issues
- Delete issues
- Simple and clean UI
- Full frontend ↔ backend integration


## 📡 API Endpoints

| Method | Endpoint        | Description        |
|--------|----------------|--------------------|
| GET    | /issues        | Get all issues     |
| POST   | /issues        | Create new issue   |
| DELETE | /issues/:id    | Delete an issue    |
---

##  Architecture
- React handles UI and user interaction
- Rails handles API requests and database logic
- Communication via REST APIs
